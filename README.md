<p align="center">This is my mini project to provide a portal to manage Mikrotik RouterOS for my hotspot. The reason behind this project is because my users got session lock on Mikrotik, but the session on their own devices has been unlinked</p>

## Features
- Support RouterOS.
- Component HTML Template.
- Simple Code.
